## Project Title
Starter for backend on nest.js

## Requirements
* Docker
* Node.js: ">=16.18.0"
* Npm: ">=6.0.0"

## Before Installation
All environment settings in .env.example
```bash
$ docker-compose up
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
